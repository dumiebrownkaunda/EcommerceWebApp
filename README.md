# EcommerceWebApp

Renamed to Ultrastore is an e-commerce web application that allows the management of users and products and the processing of products from the time of order initiation by a logged in customer,processed in the background and finalised by the administrator. 

Functionality of UltraStore:
1. User management(creation,reading,upating and deleting -CRUD)
2. Product management (CRUD)
3. Displaying of products
4. Email notifications

The projects implements the following:

-Using ASP NET MVC Core 3.
-Integrate Identity Framework.
-Interact with Razor class library for Identity.
-Integrate Entity Framework along with code first migrations.
-Sessions in ASP NET Core 3.
-Custom Tag Helpers in ASP NET Core 3.
-View Components and Partial Views in ASP NET Core.
-Email configuration
-Authentication and Authorization in ASP NET Core 3
-Deploying website on Microsoft Azure, IIS and third party hosting
-User management in ASP NET Core 3
